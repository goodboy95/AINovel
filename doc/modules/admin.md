# 模块说明：后台管理系统

## 入口与权限
- 路由前缀：`/admin`，仅管理员角色可访问。
- 布局：深色主题 AdminLayout，包含侧边导航与统计概览。

## 主要页面
- **仪表盘**：展示用户增长、积分消耗、API 健康度等指标概览。
- **模型管理**：配置模型倍率、启用状态与 API 池绑定。
- **用户管理**：查看用户列表、封禁/解封、手动发放积分。
- **积分日志**：查询积分变动明细与原因。
- **兑换码管理**：创建/查看兑换码状态与过期时间。
- **邮件验证**：查看最近验证码记录、发送 SMTP 测试邮件（用于排查注册验证码问题）。
- **系统设置**：注册开关、维护模式、签到积分区间等全局配置。

## 关键交互
- 管理操作后刷新列表状态。
- 对危险操作（封禁、发分）进行二次确认提示。
